1. Что такое «чёрные линии» в контексте рукьи?
- [ ] Видимые полосы на коже
- [ ] Линии на ладони
- [x] Духовные блокировки — раны души, через которые джинн, колдовство или сглаз влияют на конкретные сферы жизни
- [ ] Энергетические каналы в теле

2. Сколько категорий чёрных линий описано в уроке?
- [ ] Две — финансовые и духовные
- [ ] Три — тело, душа, разум
- [x] Четыре — финансовые (8), духовно-психологические (12), линии веры (13), линии тела (10)
- [ ] Одна — общая категория

3. Какой пример финансовой «чёрной линии»?
- [ ] Боль в колене
- [x] Блокировка ризка (удела) — колдовство, закрывающее двери заработка, бараки в деньгах
- [ ] Плохое зрение
- [ ] Бессонница

4. Какие способы применения дуа от чёрных линий описаны?
- [ ] Только в намазе
- [ ] Только на воду
- [x] В намазе (в суджуде), вне намаза, на воду, на оливковое масло
- [ ] Только на бумагу

5. Почему для каждой линии нужно отдельное дуа, а не одно общее?
- [ ] Потому что так красивее
- [x] Потому что спецификация (точное указание проблемы) многократно повышает эффективность заклинания
- [ ] Потому что иначе Аллах не примет
- [ ] Нет разницы — можно одним общим

6. Какой пример «линии веры»?
- [ ] Линия бедности
- [ ] Линия головной боли
- [x] Блокировка хушу' (смирения в намазе) или блокировка тауба (покаяния)
- [ ] Линия памяти

7. Сколько линий тела описано в уроке?
- [ ] 5
- [x] 10
- [ ] 20
- [ ] 3
