1. Что такое басира (بَصِيرَة) согласно уроку?
- [ ] Физическое зрение, обострённое практикой
- [ ] Способность видеть джиннов невооружённым глазом
- [x] Духовное зрение — способность сердца распознавать смыслы, знаки и мудрость Аллаха
- [ ] Особый дар, данный только пророкам

2. Какой аят из суры аль-Хадж (22:46) раскрывает разницу между физическим и духовным зрением?
- [ ] «Неужели они не размышляют над Кораном?»
- [x] «Ведь слепнут не глаза, а слепнут сердца, которые в груди»
- [ ] «Поистине, в этом — знамения для людей размышляющих»
- [ ] «Скажи: Таков мой путь. Я призываю к Аллаху на основе знания»

3. Что сказал Пророк ﷺ о фирасе (проницательности) верующего согласно хадису ат-Тирмизи (3127)?
- [ ] «Фираса — это дар, который нельзя развить»
- [ ] «Не доверяйте фирасе, она от шайтана»
- [x] «Остерегайтесь фирасы верующего, ибо он смотрит светом Аллаха»
- [ ] «Фираса приходит только после 40 лет поклонения»

4. Как связаны якын (убеждённость) и басира?
- [ ] Якын заменяет басиру — при крепком якыне басира не нужна
- [ ] Басира приходит первой и ведёт к якыну
- [x] Чем крепче якын — тем острее басира, чем чище басира — тем глубже якын
- [ ] Они не связаны между собой — это разные навыки

5. Какой из перечисленных способов развития басиры НЕ упомянут в уроке?
- [ ] Очищение сердца через тауба (покаяние)
- [ ] Размышление над Кораном (тадаббур)
- [x] Длительный пост в одиночестве
- [ ] Ночная молитва (тахаджжуд)

6. Как отличить истинную басиру от ложной, согласно уроку?
- [ ] Истинная басира всегда сопровождается физическими ощущениями
- [x] Проверять Кораном и Сунной — любое «видение», противоречащее им, от шайтана
- [ ] Истинная басира приходит только во сне
- [ ] Если после «прозрения» чувствуешь радость — значит, это истина

7. Что является верным признаком ложной басиры?
- [ ] Чувство смирения и благодарности Аллаху после «прозрения»
- [ ] Желание проверить полученное понимание через Коран и Сунну
- [x] Чувство гордости — «я вижу то, что другие не видят»
- [ ] Обращение к учителю за подтверждением

8. Согласно аяту суры Юсуф (12:108), на основе чего призывает к Аллаху Пророк ﷺ и его последователи?
- [ ] На основе чудес (му'джизат)
- [ ] На основе традиции предков
- [x] На основе знания (басиры)
- [ ] На основе страха перед наказанием
