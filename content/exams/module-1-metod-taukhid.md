1. Из каких трёх шагов состоит метод уединения с таухидом (тауxид бит-тахайюль)?
- [ ] Чтение, размышление, запоминание
- [x] Представление качества в творениях, отрицание его совершенства у них, утверждение совершенства только для Аллаха
- [ ] Дуа, зикр, намаз
- [ ] Изучение имени, повторение его 100 раз, медитация

2. При применении метода уединения к имени «Ас-Сами'» (Всеслышащий), что именно отрицается?
- [ ] Что творения вообще не слышат
- [x] Что кто-либо из творений обладает совершенным и безграничным слухом
- [ ] Что Аллах слышит только верующих
- [ ] Что звуки существуют в природе

3. Какой вид таухида укрепляется через метод уединения с именами Аллаха?
- [ ] Таухид ар-Рубубия (единобожие в Господстве)
- [ ] Таухид аль-Улюхия (единобожие в Поклонении)
- [x] Таухид аль-Асма ва-с-Сифат (единобожие в Именах и Качествах)
- [ ] Таухид аль-Хакимия (единобожие в Законодательстве)

4. Согласно уроку, какой аят запрещает попытки представить образ Аллаха?
- [ ] «Поклоняйся Аллаху и не приобщай к Нему сотоварищей» (ан-Ниса, 36)
- [x] «Нет ничего подобного Ему, и Он — Слышащий, Видящий» (аш-Шура, 42:11)
- [ ] «А те, которые уверовали, любят Аллаха сильнее» (аль-Бакара, 165)
- [ ] «Они взывают к своему Господу со страхом и надеждой» (ас-Саджда, 16)

5. Что включает Таухид аль-Улюхия (единобожие в Поклонении)?
- [ ] Признание, что только Аллах — Творец и Управитель
- [x] Направление всех видов поклонения (молитва, дуа, жертвоприношение, упование) только Аллаху
- [ ] Подтверждение имён и атрибутов Аллаха без искажений
- [ ] Изучение 99 имён Аллаха наизусть

6. Какие четыре вида уединения Аллаха в душе и сердце разобраны в уроке?
- [x] Ихляс (искренность), Тафвид (упование), Хубб (любовь), Хауф ва Раджа (страх и надежда)
- [ ] Зикр, тадаббур, тахаджжуд, садака
- [ ] Салят, закят, саум, хадж
- [ ] Сабр (терпение), шукр (благодарность), рида (довольство), зухд (аскетизм)

7. Пророк (мир ему и благословение Аллаха) сказал, что ширк в его общине «более скрытен, чем…» — что?
- [ ] Чем шёпот ветра в пустыне
- [ ] Чем тень в полуденном солнце
- [x] Чем ползёт муравья по чёрному камню в тёмную ночь
- [ ] Чем капля воды в океане

8. Что является целью воображения в методе уединения с таухидом?
- [ ] Представить сущность Аллаха для лучшего понимания
- [ ] Визуализировать Рай и Ад для усиления страха

- [ ] Развить способности ясновидения
