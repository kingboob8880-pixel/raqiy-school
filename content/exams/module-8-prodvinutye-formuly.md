1. Чем заклинание на действие отличается от обычного заклинания?
- [ ] Ничем
- [x] Обычное заклинание — «от» (мин), заклинание на действие — «для/чтобы» (ли): вывести, сжечь, развязать, извлечь
- [ ] Заклинание на действие тише
- [ ] Заклинание на действие читается на арабском, обычное — на русском

2. Какие два варианта заклинания на действие описаны?
- [ ] Длинный и короткий
- [x] С частицей «ли» (لِـ) и без неё — оба направляют действие, но «ли» делает его более целенаправленным
- [ ] Утренний и вечерний
- [ ] Мужской и женский

3. Что такое «Действие 1 — Извлечение» (хурудж)?
- [ ] Физическое извлечение предмета
- [x] Формула, направленная на выведение поражения из тела — «чтобы вышло»: сглаз, съеденное/выпитое колдовство, джинн, каринах, 'ашик
- [ ] Извлечение информации у пациента
- [ ] Хирургическая операция

4. Для каких типов поражения есть отдельные формулы извлечения?
- [ ] Только для колдовства
- [ ] Только для джинна
- [x] Для сглаза, съеденного колдовства, выпитого колдовства, джинна, каринах, 'ашик-джинна — каждому типу своя формула
- [ ] Для всех типов одна формула

5. Какой принцип лежит в основе всех продвинутых формул?
- [ ] Повторение одного аята много раз
- [x] Спецификация: чем точнее указано действие + тип поражения + орган, тем мощнее формула
- [ ] Громкость голоса
- [ ] Количество присутствующих

6. Почему формула «ли хурудж ас-сихр аль-ма'куль мин аль-ма'ида» сильнее, чем просто «мин ас-сихр»?
- [ ] Потому что она длиннее
- [x] Потому что она указывает конкретное действие (вывести), конкретный тип (съеденное колдовство), конкретный орган (желудок)
- [ ] Потому что она на арабском
- [ ] Нет разницы в силе
