1. В чём суть метода «Аллязи» (الَّذِي) в продвинутом заклинании?
- [ ] Замена имени Аллаха на имя заклинателя
- [x] Введение конкретного атрибута Аллаха из Корана через связку «Который», усиливая заклинание
- [ ] Чтение суры аль-Ихлас перед каждым заклинанием
- [ ] Повторение слова «Аллязи» сто раз

2. Какой атрибут Аллаха из суры ан-Нур (24:35) особенно силён против тьмы колдовства и васвасы?
- [ ] Аль-Хайй (Живой)
- [x] Нурус-самавати валь-ард (Свет небес и земли)
- [ ] Халику кулли шай' (Создатель всякой вещи)
- [ ] Аль-Кадир (Мощный)

3. Чем отличается заклинание на действие (с «ли» — لِـ) от обычного заклинания (с «мин» — مِنْ)?
- [ ] Ничем не отличается
- [ ] Заклинание на действие тише по голосу
- [x] «Мин» — заклинание «от» (защита), «ли» — заклинание «для/чтобы» (активное действие: вывести, сжечь, развязать)
- [ ] «Ли» используется только для воды

4. Какой атрибут Аллаха из суры аль-Ан'ам (6:96) — «Расщепляющий рассвет» — особенно силён против узлов колдовства?
- [x] Фалику-ль-исбах
- [ ] Нурус-самавати валь-ард
- [ ] Аль-Хайй аль-Каййум
- [ ] Халику кулли шай'

5. Что такое комбинированная формула максимальной мощи?
- [ ] Чтение трёх разных сур подряд
- [ ] Заклинание без атрибутов
- [x] Формула, сочетающая атрибут Аллаха + «от» (что убирается) + «для» (что происходит вместо этого)
- [ ] Чтение одного аята сто раз

6. Какое условие автор называет первым для эффективности метода «Аллязи»?
- [ ] Громкий голос
- [ ] Наличие свидетелей
- [x] Иман и якын — убеждённость, что исцеляет только Аллах, а ты лишь причина
- [ ] Знание арабской грамматики
