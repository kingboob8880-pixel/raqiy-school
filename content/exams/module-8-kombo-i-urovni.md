1. Что такое комбо-формула (цепная атака)?
- [ ] Чтение одной формулы громко
- [x] Цепочка из нескольких действий подряд без паузы — например: развязать узлы + сжечь колдовство + извлечь + запечатать
- [ ] Одна формула, повторённая 100 раз
- [ ] Чтение двумя заклинателями одновременно

2. Сколько типов действий (Action Types) перечислено в таблице?
- [ ] 5
- [x] 10
- [ ] 3
- [ ] 15

3. Какие три уровня мастерства выделяет автор?
- [ ] Студент, учитель, мастер
- [x] Начинающий (мубтади'), практикующий (мумарис), мастер/сильнейший (аква)
- [ ] Слабый, средний, сильный
- [ ] Ученик, подмастерье, гроссмейстер

4. Чем уровень «Мастер» отличается от уровня «Начинающий»?
- [ ] Только количеством прочитанных книг
- [x] Мастер использует комбо-формулы с максимальной спецификацией, все 8 компонентов на максимуме, минимум повторений для результата
- [ ] Мастер читает громче
- [ ] Нет принципиальной разницы

5. Какой пример комбо-формулы против завязанного колдовства?
- [ ] Только «мин ас-сихр»
- [x] Развязать узлы + аннулировать колдовство + извлечь остатки + запечатать — цепочка из четырёх действий
- [ ] Прочитать Аят аль-Курси один раз
- [ ] Дуть на воду

6. Что автор говорит о переходе между уровнями?
- [ ] Переход происходит автоматически с возрастом
- [ ] Нужен экзамен у шейха
- [x] Переход — через практику: чем больше реальных сеансов, тем выше точность, скорость и сила; без практики уровень не растёт
- [ ] Достаточно прочитать все книги

7. Какова роль полной комбо-формулы уничтожения (Full Destruction Combo)?
- [ ] Используется только при обучении
- [x] Максимально мощная цепная атака против самых упорных поражений — когда отдельные формулы не дают результата
- [ ] Читается только мастером
- [ ] Применяется только к ифритам
