1. Что такое «грязь души» согласно уроку?
- [ ] Физическая нечистота тела, требующая омовения
- [x] Непрожитые, подавленные эмоции (страх, обида, гнев, вина), которые «записываются» в теле и становятся почвой для духовных поражений
- [ ] Грехи, которые смываются только хаджем
- [ ] Наущения шайтана, которые приходят во сне

2. Как работает метод диагностики «телесный отклик»?
- [ ] Лекарь осматривает тело пациента и находит видимые изменения
- [ ] Пациент описывает свои сны, и лекарь определяет загрязнение
- [x] Сознательное проговаривание слова-триггера (например, «страх»), затем сканирование тела для обнаружения отклика в конкретной зоне
- [ ] Чтение Корана над пациентом и наблюдение за его реакцией

3. Где, согласно таблице урока, обычно проявляется обида?
- [ ] В пояснице и почках
- [x] В горле и груди (область сердца) — ком в горле, сдавленность, камень на сердце

- [ ] В печени и висках

4. По какому шаблону строятся все формулы заклинания от недугов души?
- [ ] А'узу биЛляхи мин + [недуг]

- [ ] Аллахумма-шфини мин + [недуг]
- [ ] Ля хауля ва ля куввата илля биЛлях + [недуг]

- [ ] «Есть в теле кусочек плоти — это сердце» (аль-Бухари, 52)
- [x] «Когда раб совершает грех — на его сердце ставится чёрная точка» (ат-Тирмизи; Ибн Маджа)

- [ ] «Поистине, Аллах смотрит на ваши сердца» (Муслим, 2564)
