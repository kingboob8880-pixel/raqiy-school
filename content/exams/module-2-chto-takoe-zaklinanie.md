1. Каковы пять качеств истинного заклинателя согласно уроку?
- [ ] Знание, терпение, мудрость, щедрость, скромность
- [x] Якын (убеждённость), Ихляс (искренность), Шаджа'а (бесстрашие), Таркиз (точность), Амаль (действие)
- [ ] Иман, Ислам, Эхсан, таваккуль, сабр
- [ ] Тауба, зикр, намаз, пост, садака

2. Что означает Хитаб (الخِطَاب) и Исаба (الْإِصَابَة) в системе заклинания?
- [ ] Хитаб — это текст заклинания, Исаба — его результат
- [x] Хитаб — обращение и направленность (к кому, каким голосом, куда), Исаба — попадание и точность (конкретика формулировки, физическое сопровождение, визуализация)
- [ ] Хитаб — громкость голоса, Исаба — скорость чтения
- [ ] Хитаб — выбор суры, Исаба — количество повторений

3. Какой аят подтверждает, что заклинание должно совершаться с полной искренностью (ихляс)?
- [x] «И не было им велено, кроме как поклоняться Аллаху, искренне посвящая Ему религию» (аль-Байина, 98:5)
- [ ] «Нет ничего подобного Ему» (аш-Шура, 42:11)
- [ ] «Поистине, дела — по намерениям» (аль-Бухари, 1)
- [ ] «Козни шайтана слабы» (ан-Ниса, 4:76)

4. Почему точность и направленность (Таркиз) важны в заклинании?
- [ ] Чтобы произвести впечатление на пациента
- [ ] Чтобы заклинание звучало красивее
- [x] Чем конкретнее указан недуг и место поражения, тем сильнее воздействие — размытая мольба как стрела без мишени
- [ ] Чтобы не забыть текст заклинания

5. Какие физические действия должны сопровождать заклинание согласно Сунне Пророка ﷺ?
- [ ] Только чтение вслух без дополнительных действий
- [ ] Раскачивание тела и хлопки в ладоши
- [x] Дуть на больного, положить руку на место боли, дышать на воду при чтении
- [ ] Рисовать знаки на теле больного

6. Какая из перечисленных причин НЕ названа в уроке как причина неработающего заклинания?
- [ ] Слабая вера — сомнения «а вдруг не сработает?»
- [ ] Нечистые помыслы — желание похвастаться силой
- [x] Неправильный тажвид при чтении аятов
- [ ] Отвлечённое сердце — язык произносит, а сердце думает о другом

7. Согласно аяту ан-Ниса (4:76), каковы козни шайтана?
- [ ] Непобедимы для обычного человека
- [ ] Равны по силе козням людей
- [x] Слабы — тот, кто уповает на Аллаха, сильнее любого колдовства
- [ ] Действуют только на неверующих

8. Как должен звучать голос целителя при разрушении колдовства в отличие от голоса при исцелении?
- [ ] Одинаково тихо и спокойно в обоих случаях
- [ ] Одинаково громко и резко в обоих случаях
- [x] Твёрдый и чёткий — для разрушения колдовства, тёплый и мягкий — для исцеления
- [ ] Шёпотом при разрушении, громко при исцелении
