1. По каким трём измерениям автор сравнивает практика и теоретика?
- [ ] Возраст, опыт, образование
- [x] Глубина понимания, подход к решению, диагностика
- [ ] Сила голоса, количество формул, скорость чтения
- [ ] Знание арабского, знание хадисов, знание фикха

2. Как практик отвечает на вопрос «Что делать, если пациенту стало плохо во время сеанса?»
- [ ] «Прочитай аят аль-Курси»
- [x] Конкретный алгоритм: остановить чтение → проверить состояние → возобновить с другой формулой / сменить орган
- [ ] «Позвони врачу»
- [ ] «Прекрати сеанс и уходи»

3. Какой аят приводится в подтверждение разницы между знающим и незнающим?
- [ ] Аль-Бакара, 2:255
- [x] Сура аз-Зумар, 39:9 — «Скажи: разве равны те, которые знают, и те, которые не знают?»
- [ ] Сура аль-Фатиха
- [ ] Сура аль-'Аср

4. Чем диагностика практика отличается от диагностики теоретика?
- [ ] Практик использует приборы
- [ ] Теоретик точнее
- [x] Практик определяет поражение по словам пациента, по признакам и по реакции на чтение; теоретик опирается только на книжные описания
- [ ] Нет разницы

5. Какой вывод делает автор о знании без практики?
- [ ] Знание без практики — это полный путь
- [ ] Практика без знания лучше, чем знание без практики
- [x] Знание без практики — половина пути; практика без знания — опасность
- [ ] Знание и практика не связаны

6. Какой практический пример показывает разницу в подходе к решению?
- [ ] Выбор между двумя книгами
- [x] Теоретик скажет «нужна рукья от колдовства», практик определит точный вид колдовства, орган, формулу и построит план на 7 дней
- [ ] Теоретик читает быстрее
- [ ] Практик отказывается от сложных случаев
