1. Кто назван «аль-Хакк» (Истина) в Коране?
- [ ] Пророк Мухаммад ﷺ
- [ ] Ангел Джибриль
- [x] Аллах — сура аль-Хадж, 22:62: «Это потому, что Аллах есть Истина»
- [ ] Коран

2. Почему Коран называется истиной (аль-хакк)?
- [ ] Потому что он написан на арабском
- [ ] Потому что он самая длинная книга
- [x] Потому что он ниспослан от Аллаха, Который Сам есть Истина, и подтверждает предыдущие Писания (аль-Маида, 5:48)
- [ ] Потому что его много читают

3. Какой аят подтверждает, что Сунна — это разъяснение истины, а не слова от себя?
- [ ] Аль-Бакара, 2:255
- [x] Сура ан-Наджм, 53:3–4 — «Он не говорит по прихоти. Это лишь откровение, которое внушается»
- [ ] Сура аль-Ихлас
- [ ] Сура Ясин, 36:1

4. Что является фундаментом истины в исламе?
- [ ] Хадисоведение
- [ ] Арабский язык
- [x] Таухид — единобожие, признание Аллаха единственным Богом, без сотоварищей
- [ ] Знание фикха

5. Какова связь между истиной и рукьей?
- [ ] Рукья — это магия, не связанная с истиной
- [x] Рукья эффективна только когда основана на истине: Коран (истина от Аллаха) + Таухид (фундамент) + Сунна (метод)
- [ ] Связи нет — рукья работает автоматически
- [ ] Истина — это теория, рукья — практика, они не пересекаются

6. Почему заклинатель, отступивший от истины, теряет силу?
- [ ] Потому что джинны его не боятся
- [x] Потому что сила рукьи идёт от Аллаха через Коран и Таухид; отступление от истины разрывает эту связь
- [ ] Потому что он забывает формулы
- [ ] Потому что пациенты ему не доверяют
